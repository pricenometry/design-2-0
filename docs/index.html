File structure

dest Main folder consists 6 predefined static sites blue, dark_turquoise, green, orange, pink and purple. Also you can find "index.html" file for start demo.
docs Documentation folder
licenses Licenses files
src Handlebars sources are located here. This folder is used for generate static sites to "dest" folder.
package.json config file for npm package manager
Gruntfile.js onfig file for Grunt

  • main.min.css - Minified Main file of stylesheets. Use can use unminified "css/main.css" file

Compressed scripts

You can include only 1 script instead of all. So comment all scripts and add next line:

  
        <script src="js/compressed.min.js"></script>

How to install node, npm and grunt

For working with sources you shold do next:

1. Install Node.js with npm

2. Install Grunt.js

3. Run "npm install" in console

After this you can use grunt tasks

Grunt tasks

1. Command "grunt minify_js" for minify and concat js files

2. Command "grunt build" for build all static sites to desc folder. All files in desc folder will be rewrited

3. Command "grunt" is alias for "grunt build"
